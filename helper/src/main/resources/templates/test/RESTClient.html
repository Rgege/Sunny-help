数据迁移助手（从RESTClient2迁移）

                
            

            

                
                
    
                    
  • 
                        操作说明
                    
    • 
                    
  • 
                        迁移
                    
    • 
                


                
                

                    

                        
Firefox
                            57 开始不再支持老式 Firefox 插件，同时对插件的权限进行了限制。当前 Firefox 插件不再具备访问 Firefox
                            偏好设置的能力。而RESTClient 2中的所有数据均存储在 Firefox 偏好设置中。因此您需要手动迁移 RESTClient 2
                            的数据到 RESTClient 3 中。

                        
以下内容将介绍如何使用 about:config 页面获得
                            RESTClient 2 的相关数据。

                        
第一步： 打开 about:config
                            页面

                        
在浏览器的地址栏中输入：about:config 并回车。
                            
 about:config 页面会提醒您“这可能导致质量保证失效！”， 点击下方“我了解此风险”按钮继续。

                        

                            
                        

                        
第二步：搜索 RESTClient
                            的偏好设置，并且复制这些值

                        
请在 about:config 页面顶部的 Search
                            栏中输入 restclient 来搜索 RESTClient2 相关的偏好设置。

                        

                            
                        

                        
第三部：打开本迁移助手的 迁移
                            标签页面，粘贴您拷贝的数据

                        
您需要从 about:config 页面复制以下数据：
                        
    
                            
  • extension.restclient.cachedUrls
    • 
                            
  • extension.restclient.favoriteHeaders
    • 
                            
  • extension.restclient.savedRequest
    • 
                        

                        

                        

                            
                        

                    

                    

                        
                        

                            
                            

                                
                                粘贴 `extension.restclient.cachedUrls`
                                    的值。
                                
                            

                        

                        

                            
                            

                                
                                粘贴 `extension.restclient.favoriteHeaders`
                                    的值。
                                
                            

                        

                        

                            
                            

                                
                                粘贴 `extension.restclient.savedRequest`
                                    的值。
                                
                            

                        

                        

                            

                            

                                
                            

                        

                        
                    

                


            

            

                
                
            

        

    





    

        

            

                
null 的证书不合法

                
            

            
            

                

                    网站「null」所使用的证书不是合法的证书。请在新窗口中打开该网站页面，在你警告页点击「高级」按钮，点击「添加例外」按钮，并在添加安全列外对话框中点击「确认安全例外」。在将您的网站「null」加入例外后，您可以在
                    RESTClient 中继续测试您的网站。

                

                    
                

                
了解更多信息，请访问 Firefox 官方网站：如何针对安全网站的错误码进行故障排除

您当前 HTTP 请求的返回结果类型为：「null」，为正确显示该 HTTP
                    响应，您需要将 HTTP 响应类型设置为「文件型（Blob）」。
